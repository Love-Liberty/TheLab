📦 client.html

This file defines and exports a Supabase client instance, used throughout the app for database interactions. It centralizes configuration and ensures consistent access to Supabase services.

🔍 Summary

PurposeInitialize and export a Supabase client
InputsSupabase URL and public key (usually from environment or config)
OutputsConfigured Supabase client object
DependenciesSupabase JS SDK

🧠 Design Notes

  • Encapsulates setup logic to avoid duplication across modules
  • Can be extended later to include auth headers or session management
  • Should be imported wherever database access is needed
