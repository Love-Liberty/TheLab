🔄 changePage.html

This file manages pagination logic — switching between pages of notes or categories. It updates the UI and triggers data reloads based on the current page state.

🔍 Summary

PurposeHandle pagination and page switching
InputsCurrent page index, user navigation actions
OutputsUpdated UI and reloaded data
DependenciesdisplayNotes, DOM elements, page state

🧠 Design Notes

  • Supports forward/back navigation and direct page jumps
  • Can be extended to show page numbers or progress indicators
  • Should debounce rapid clicks to avoid redundant reloads
