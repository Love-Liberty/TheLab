🏷️ collectUserChoices.js

Parses checked inputs from the tag UI and returns selected values.

Suggested Implementation


export function collectUserChoices() {
  const tags = new Set();
  const inputs = document.querySelectorAll('input[type="checkbox"]:checked, input[type="radio"]:checked');
  inputs.forEach(input => tags.add(input.value));
  return tags;
}

Notes

  • Supports multiple tag groups
  • Returns a flat Set of tag values
  • Could be extended to preserve tag categories
